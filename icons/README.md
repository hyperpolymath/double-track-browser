# Extension Icons

This directory contains the icon assets for DoubleTrack Browser.

## Required Sizes

The extension requires icons in the following sizes:
- 16x16 - Toolbar icon
- 32x32 - Toolbar icon (retina)
- 48x48 - Extension management page
- 128x128 - Chrome Web Store

## Creating Icons

To create proper icons, use a tool like:
- Figma
- Adobe Illustrator
- Inkscape (free)

Design guidelines:
- Use a simple, recognizable symbol
- Consider a "double track" or "parallel lines" motif
- Colors should work well on light and dark backgrounds
- Maintain good contrast for visibility

## Placeholder Icons

Currently using placeholder icons. Replace these with proper designed icons before release.

## Icon Theme

Suggested theme: Two parallel paths or tracks, representing the dual nature of browsing (real + fictional). Colors could include blues and purples matching the extension's UI theme.
